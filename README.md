# China Narrative Tracker

An NLP-based media analysis tool that tracks how China is portrayed across international news outlets over time.

## Research Questions

- How has China's portrayal in international media changed over time?
- Do Western and Asian media frame China differently?
- How have COVID-19 and the US-China trade war influenced media narratives?

## Planned Features

- News article collection
- Sentiment analysis
- Framing analysis
- Topic modeling
- Narrative detection
- Historical media comparison
- Interactive dashboard

## Tech Stack

- Python
- SQLite
- NewsAPI
- spaCy
- Transformers
- Plotly
- Streamlit
